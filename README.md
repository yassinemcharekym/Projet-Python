# Projet – Chiffrement et Déchiffrement en Python

## Objectif
L’objectif de ce projet est de chiffrer et déchiffrer des fichiers contenant des données sensibles
à l’aide d’un script Python.

## Technologie utilisée
- Langage : Python 3
- Bibliothèque : cryptography (Fernet)
- Type de chiffrement : symétrique

## Principe
Une clé secrète est générée lors du premier chiffrement.
Cette clé est utilisée pour chiffrer et déchiffrer les fichiers.
